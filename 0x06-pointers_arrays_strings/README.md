# C Pointers, Arrays And Strings
This project focuses on the use of pointer, arrays and strings in C.

| __File__| __Description__ |
|--------------|-------------|
|`0-strcat.c`| concatenates two strings.|
|`1-strncat.c`| concatenates a string and n bytes of another string.|
|`2-strncpy.c`| copies a string.|
|`3-strcmp.c`| compares two strings.|
|`4-rev_array.c`| reverses the content of an array of integers|
|`5-string_toupper.c`| changes all lowercase letters of a string to uppercase.|
|`6-cap_string.c`| capitalizes all words of a string.|
|`7-leet.c`|encodes a string into 1337|
|`100-rot13.c`|encodes a string using rot13|
|`101-print_number.c`|prints an integer|
|`102-magic.c`|Changes the value of an index of an integer array|
|`103-infinite_add.c`|adds two numbers|
|`104-print_buffer`|prints a buffer|
|`holberton.h`|header file containing prototypes used in this project|
